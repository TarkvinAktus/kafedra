# Information system for university chair
It is my diploma project. Theme for this project have been choosen for get more practice in web development. 
This web application designed to service university chair's internal processes and solves the next problems.

  - News blog for university chair
  - Personal pages for students and teachers
  - Provide structured storage for learning documents
  - Journal for personal tasks and projects
  - Journal for student's diploma projects


In this project I've used python 3.7 with framework django 2.2 for backend part and simple bootstrap pages for frontend part.
